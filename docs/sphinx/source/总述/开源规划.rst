========================================================================
开源规划
========================================================================

Bi-Map将分为三个阶段逐步开源文档、代码，三个阶段的划分如下图所示。

.. image::  _static/plan1.png
   :width: 65%
   :align: center


在开源过程中，我们会在测试库中不断地加入回归测试用例，使得Bi-Map不断变得稳定。因为硬件一致层涉及很多芯片核心技术细节，所以不会开源，但可能在之后考虑提供开放使用的途径。
